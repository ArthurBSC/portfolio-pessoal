Sobre mim

Sou um Desenvolvedor com 20 anos de idade, estudante de Engenharia de Software na UNIALFA. Sempre fui curioso sobre tecnologia e em 2023 mergulhei de cabeça na programação, descobrindo minha paixão pelo desenvolvimento.

Atualmente trabalho como desenvolvedor, aplicando conhecimentos em JavaScript, React, TypeScript, Node.js, PHP, Docker, HTML, CSS e banco de dados. Desenvolvi projetos como o sistema BarbaBranca para barbearia e o Nexa - IA com 5 modelos treinados usando Ollama e Llama 2.0, demonstrando minha capacidade de criar soluções práticas e inovadoras.

O que eu faço

  • Desenvolvimento Front-end

    Criação de interfaces modernas e responsivas usando React, TypeScript, JavaScript, HTML e CSS.

  • Desenvolvimento Back-end

    Desenvolvimento de APIs e sistemas robustos usando Node.js, PHP, MySQL e PostgreSQL.

  • Containerização

    Implementação de soluções com Docker para deploy e gerenciamento de aplicações.

  • Desenvolvimento Full Stack

    Desenvolvimento completo de aplicações web usando JavaScript, TypeScript, React, Node.js e PHP.
